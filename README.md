
## Setup

1. Установить все необходимые зависимости

```bash
npm install
```
2. Отдельно сбилдить папку для nuxt. В консоли могут быть ошибки связынные с билдом миксинов для scss - ЭТО НЕСТРАШНО, так как они все равно работают

```bash
npm run generate
```

3. Запускаем dev сервер. В консоли могут быть ошибки связынные с билдом миксинов для scss - ЭТО НЕСТРАШНО, так как они все равно работают

```bash
npm run dev
```
